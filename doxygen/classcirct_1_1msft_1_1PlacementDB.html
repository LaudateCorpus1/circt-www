Detailed Description

A data structure to contain both the locations of the primitives on the device and instance assignments to said primitives locations, aka placements.

Holds pointers into the IR, which may become invalid as a result of IR transforms. As a result, this class should only be used for analysis and then thrown away. It is permissible to persist it through transformations so long as it is maintained along with the transformations.
